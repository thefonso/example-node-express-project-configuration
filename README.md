## Instructions

Download the code:

1. Press the green "code" button
2. Choose the "Https" option and copy this line "https://github.com/Thinkful-Ed/example-node-express-project-configuration.git"
3. Open the terminal program on your local computer and enter: git clone https://github.com/Thinkful-Ed/example-node-express-project-configuration.git
4. cd into the newly created folder and open in your editor of choice.

Install and run:

1. After you cd into the new folder, run `npm install`
2. And then run it with `npm start`
